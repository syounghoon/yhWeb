# FILMUS Project

## ■ Info

- #### EC2 연결 [**연결**](http://52.79.71.243:8090/)
- #### 전체 포트폴리오 [**보기**](https://younghoon.s3.ap-northeast-2.amazonaws.com/Filmus/portpolio.pdf)


#### My Part
 - AWS EC2 기반으로 프로젝트 구현
 - AWS S3 사용하여 파일 업로드 다운로드 구현
 - 메인
 - 영화/유저 검색
 - 검색결과
 - 마이페이지
   - 팔로우 기능
   - 활동로그 기능
 - 회원정보 변경


___

## ■ About Project
- '영화'를 중심으로 함께 소통할 수 있는 공간을 만들기 위해 기획 / 제작
- 영화를 좋아하는 사람들의 니즈 파악
- 서로 리뷰를 공유하고 좋아하는 리뷰어를 팔로우하는 SNS기능이 가미된 리뷰 공유 사이트

___
